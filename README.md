christmastree
=============

Christmas tree in pure JAVA and console.
``` bash
          *      
         '~.     
        .~'O'    
       ~..~'O'   
      ~..~'O'~.  
     .~'O'~..~'O 
    '~..~'O'~..~'
         |~|     
```

Usage :
> mvn clean install

> cd target

> java -jar christmastree-1.0-SNAPSHOT.jar 6

where 6 - is tree height will output christmas tree like above
